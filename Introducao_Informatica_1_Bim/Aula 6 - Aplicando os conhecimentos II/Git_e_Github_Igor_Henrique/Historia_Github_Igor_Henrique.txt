O Git é um projeto de código aberto maduro e com manutenção ativa desenvolvido em 2005 por Linus Torvalds, o famoso criador do kernel do sistema operacional Linux.
O Git foi criado com a proposta de resolver a necessidade de um sistema de distribuição descentralizada, e que seguisse a filosofia de software livre, assim como o Linux, e sem se preocupar em ficar preso a uma ferramenta proprietária que de um dia pro outro os criadores resolvessem comercializá-lo.
A Microsoft é a atual dona do GitHub, e o presidente-executivo da empresa, Satya Nadella afirma que,(A Microsoft é uma companhia voltada ao desenvolvedor e, ao unir forças com o GitHub, nós fortalecemos nosso comprometimento com a liberdade do desenvolvedor, abertura e inovação)
Podemos trabalhar com o GitHub através da sua plataforma online, onde podemos criar repositórios, arquivos... todos os processos que fazemos no terminal, podemos fazer lá.

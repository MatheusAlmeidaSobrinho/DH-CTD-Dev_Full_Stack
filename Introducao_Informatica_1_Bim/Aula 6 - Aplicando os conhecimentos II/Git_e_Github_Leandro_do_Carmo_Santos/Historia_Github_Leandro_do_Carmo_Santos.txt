a. Quem inventou o sistema de controle de versão Git e por quê?
   R: O Git foi inicialmente projetado e desenvolvido por Linus Torvalds
     para o desenvolvimento do kernel Linux, mas foi adotado por muitos outros projetos.

b. Quem atualmente é o dono do Github e por quê?
   R: A Microsoft. para concentrar seus esforços nas quatro principais frentes que Nadella 
      considera cruciais: Nuvem, IA, Serviços (incluso Windows, Office e Xbox) e mercado corporativo.

c. Existe outra maneira, além do terminal, de trabalhar com o Github?
   R: Sim.
